# DomoScreen
Un écran domotique pour la maison

Besoin requis:

1- Un écran tactile:

GUITION ESP32-S3-4848S040_I_Y_3 
C'est un écran de 4 pouces avec une résolution de 480x480
Celui là a 3 relais internes, mais pas forcément indipensables vu qu'avec la domotique on peut ajouter des switch wifi.
Avec LVGL(Light and Versatile Graphics Library) intégré.

2- openHASP (home automation switch plate)
https://www.openhasp.com/0.7.0/
https://github.com/HASwitchPlate/openHASP

3- home assistant intallé et fonctionnel
https://www.home-assistant.io/
